# 기도수첩 📖🙏

중보기도 기도제목 관리 PWA 앱

## 기능
- 그룹별 기도제목 관리 (그룹 추가/수정/삭제 가능)
- 기도 상태 관리 (미응답 / 진행중 / 응답됨)
- 기도 모드 (한 화면씩 집중 기도)
- CSV 불러오기 / 내보내기
- 응답받은 기도 보관함
- 다중 선택 삭제
- 오프라인 사용 가능 (PWA)

## 배포 (GitHub Pages)

1. 이 저장소를 생성합니다
2. Settings → Pages → Source: `main` 브랜치 선택
3. 저장하면 `https://{사용자명}.github.io/{저장소명}/` 으로 접속 가능

## 파일 구조
```
index.html     ← 메인 앱
manifest.json  ← PWA 설정
sw.js          ← 오프라인 서비스워커
icon-192.png   ← 앱 아이콘 (홈화면용)
icon-512.png   ← 앱 아이콘 (스플래시용)
favicon.ico    ← 브라우저 탭 아이콘
```

## 데이터 저장
모든 기도제목은 기기의 `localStorage`에 저장됩니다.  
브라우저 데이터를 지우면 삭제되므로, 정기적으로 **전체 내보내기(백업)** 를 권장합니다.

## 스마트폰 홈화면에 추가하는 방법

**안드로이드 크롬**: 우측 상단 ⋮ → "홈 화면에 추가"  
**삼성 인터넷**: ☰ → "페이지 추가" → "홈 화면"  
**아이폰 사파리**: 공유 버튼 □↑ → "홈 화면에 추가"
